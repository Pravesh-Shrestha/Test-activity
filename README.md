# Birthday Activity (React + Vite) 🎂

A beautiful, interactive birthday app built with React, Vite, and Tailwind CSS.
Featuring a 3D-styled cake, confetti, sound effects, and smooth animations.

## 🚀 How to Run Locally (For Developers)

If someone sends you this source code, follow these steps to run it:

1.  **Install Node.js** (if you don't have it): [Download Here](https://nodejs.org/)
2.  **Open Terminal** in this project folder.
3.  **Install Dependencies**:
    ```bash
    npm install
    ```
4.  **Start the App**:
    ```bash
    npm run dev
    ```
5.  **View in Browser**: Open the link shown in the terminal (usually `http://localhost:5173`).

## 🌐 How to Share as a Link (For Everyone)

The easiest way to share this is by deploying it to **GitHub Pages**.

1.  **Make sure your changes are committed**:
    ```bash
    git add .
    git commit -m "Ready for deploy"
    ```
2.  **Deploy**:
    ```bash
    npm run deploy
    ```
3.  **Share the Link**:
    Your app will be live at:

_(Note: It might take a minute or two for GitHub to update the site after you deploy.)_

## 🛠️ Project Structure

-   `src/App.jsx` - Main logic (Screens: Intro -> Wish -> Cake -> Letter).
-   `src/components/Cake.jsx` - Interactive 3D cake component.
-   `src/components/Dog.jsx` - The guide dog character.
-   `public/data.json` - Content config (Text, messages).
